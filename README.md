## âœ¨ About Magic Shell

Tired of forgetting complex commands or spending time searching for the right flags? **Magic Shell** is here to change that. By integrating powerful natural language processing directly into your terminal, it allows you to execute commands by simply describing what you want to do. It's an intuitive, beautiful, and "magical" shell designed for developers, system admins, and students alike.

---

## ðŸš€ Key Features
### Basic
1. Execute standard shell commands (ls, cd, mkdir, etc.)
2. Trigger AI assistance with a prefix (/ai) to turn plain English prompts into real commands
3. Maintain a command history for both manual and AI-generated inputs
4. Intelligent auto-completion and suggestions powered by command history and AI
5. Preview and confirm potentially dangerous commands (like rm -rf)

### Advanced
1. Context-aware: tracks your current directory, files, and environment
2. Automates multi-step tasks from a single natural language instruction
3. Create custom shortcuts and templates for frequent tasks
4. Interactive mode for clarifying ambiguous commands
5. Integrations with external APIs (e.g., Git, Docker) for extended AI capabilities
6. Cross-platform support: works on Windows, macOS, and Linux

# Installation
1.  Clone the repository to your local machine:
    ```sh
    git clone [https://github.com/Champhimu/MagicShell.git](https://github.com/Champhimu/MagicShell.git)
    ```
2.  Navigate into the project directory:
    ```sh
    cd MagicShell
    ```
3.  Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```
4.  Run the application:
    ```sh
    python app.py
    ```
---

# Usage
1.  **Run Standard Commands**: Type any normal command like `mkir`, `pwd` and press Enter.
2.  **Use the AI Helper**:
    * Prefix your request with `/ai`.
    * Describe the task you want to perform in plain English.
    * The AI will translate it into a shell command, which you can then approve to execute.
3. Use arrow keys and Tab for navigating history and auto-completions.
